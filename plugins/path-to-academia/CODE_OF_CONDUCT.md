# Code of Conduct

path to academia is a public technical project for agent-assisted academic opportunity research.
The project welcomes focused, good-faith contributions that improve reliability, documentation,
security, and generality.

## Expected Behavior

- Be precise and respectful in issues, reviews, and discussions.
- Critique code, data quality, workflow design, and evidence standards rather than people.
- Keep private applicant data, unpublished source collections, and personal outreach notes out of
  public issues and pull requests.
- Disclose security-sensitive details through the security process, not a public issue.

## Unacceptable Behavior

- Harassment, threats, discriminatory language, or sustained personal attacks.
- Publishing private personal data or private outreach notes.
- Misrepresenting unverifiable research claims as sourced facts.
- Using project channels for spam or unrelated promotion.

## Enforcement

Maintainers may edit, hide, or remove comments and may block contributors who repeatedly violate
these rules. Security and privacy issues should be handled according to [SECURITY.md](SECURITY.md).
